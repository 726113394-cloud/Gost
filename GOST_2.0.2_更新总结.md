# Gost 插件 2.0.2 版本更新总结

## 📋 更新概述

本次更新主要解决了编译错误问题，并新增了黑暗效果系统，增强了游戏的恐怖氛围。

## 🔧 修复的问题

### 1. 编译错误修复
#### PlayerManager.java - 缺少ItemMeta导入
- **问题**: `找不到符号: 类 ItemMeta`
- **修复**: 添加了 `import org.bukkit.inventory.meta.ItemMeta;`
- **位置**: 第11行

#### SecondChanceListener.java - 错误的枚举常量
- **问题**: `找不到符号: 变量 GHOST`
- **原因**: `PlayerManager.PlayerRole` 枚举中没有 `GHOST` 常量，只有 `GHOST_MOTHER` 和 `GHOST_NORMAL`
- **修复**: 使用 `isHuman()` 和 `isGhost()` 方法替换直接枚举检查
- **优化**: 代码更简洁，避免重复逻辑

#### ItemSpawnManager.java - 代码优化
- **修复**: 使用 `isHuman()` 和 `isGhost()` 方法替换直接枚举检查
- **效果**: 代码更简洁，提高可维护性

### 2. 功能问题修复
#### 灵魂探测器失效问题
- **问题**: v2.0.1版本中"灵魂探测器"道具可能失效
- **影响**: 玩家无法正常使用灵魂探测器暴露其他玩家位置
- **修复**: 修复了相关逻辑，确保灵魂探测器正常工作
- **版本**: 此问题在v2.0.2中已完全修复

## 🎮 新增功能：黑暗效果系统

### 1. 配置系统更新
- **新增配置项**:
  ```yaml
  dark-effect:
    enabled: false # 是否启用黑暗效果（给予所有玩家黑暗药水效果）
    duration: 999999 # 黑暗效果持续时间（秒），设置为极大值以持续整个游戏
    amplifier: 0 # 黑暗效果等级（0为默认）
  ```

### 2. 新增DarkEffectManager
- **功能**:
  - 管理黑暗效果的启用/禁用
  - 给所有游戏玩家应用/移除失明效果
  - 支持配置化的持续时间和效果等级

### 3. 新增管理员命令
- **命令**: `/gostadmin dark <on|off|status>`
- **功能**:
  - `on`: 启用黑暗效果，给所有玩家应用失明效果
  - `off`: 禁用黑暗效果，移除所有玩家的失明效果
  - `status`: 查看黑暗效果状态

### 4. 游戏集成
- **自动应用**: 游戏开始时自动应用黑暗效果（如果启用）
- **配置控制**: 通过config.yml控制是否启用

## 🎯 使用说明

### 启用黑暗效果
1. 编辑 `config.yml`，设置 `dark-effect.enabled: true`
2. 或使用命令 `/gostadmin dark on`

### 配置黑暗效果
- `duration`: 效果持续时间（秒），建议设置为极大值（如999999）以持续整个游戏
- `amplifier`: 效果等级，0为默认失明效果，数值越高效果越强

### 注意事项
- 黑暗效果使用 `BLINDNESS`（失明）药水效果
- 效果在游戏开始时自动应用给所有玩家
- 可以通过命令实时切换开关状态
