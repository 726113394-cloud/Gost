# Gost - 生化模式小游戏

基于Residence领地的团队对抗游戏，模拟"穿越火线"的"生化模式"玩法。

## 功能特点

### 🎮 核心玩法
- 玩家分为"人类"和"鬼"两个阵营
- 鬼在限定时间内感染所有人类
- 人类需要在限定时间内保证不被感染
- 被感染的人类会变成鬼

### 🏆 经济系统
- 入场费：100金币（可配置）
- 服务器奖金：5000金币（可配置）
- 奖池 = 所有玩家入场费 + 服务器奖金
- 人类胜利：按存活时间比例分配奖池
- 鬼胜利：按感染人数比例分配奖池

### 👥 队伍系统
- 使用原版队伍系统
- 人类：白色队伍
- 鬼：红色队伍
- 母体鬼：深红色队伍
- 游戏结束后自动清除队伍

### 🎯 道具系统（每分钟自动发放）
1. **所有人类**：肾上腺素药水（速度效果）
2. **所有鬼**：狂暴药水（速度效果）
3. **随机6位玩家**：凝冰球（减速鬼）
4. **随机3位人类**：控魂术（冻结所有鬼）
5. **随机6位玩家**：闪现珍珠（短距离传送）
6. **最后2分钟**：随机将一名鬼变回人类

### ⚡ 特殊效果
- 母体鬼：前20秒失明效果
- 每分钟：所有玩家高亮5秒（幽灵感知）
- 感染时：闪电/烟花效果 + 音效
- 固定2点生命值（需要Injured Effects模组）
- **ActionBar提示**：使用道具时在物品栏上方显示使用方法

### 🗺️ 游戏区域系统
- **选区工具**：使用岩浆膏（可配置）进行左键/右键选区
- **区域管理**：支持保存、加载、删除多个游戏区域
- **区域存档**：最多可保存20个游戏区域
- **自动传送**：自动传送到区域中心
- **边界检查**：禁止离开游戏区域

### 🛠️ 区域管理命令
- `/gostadmin tool` - 获取选区工具
- `/gostadmin pos1` - 设置第一个点（工具左键）
- `/gostadmin pos2` - 设置第二个点（工具右键）
- `/gostadmin save <名称>` - 保存当前选区
- `/gostadmin list` - 列出所有存档区域
- `/gostadmin load <名称>` - 选择区域
- `/gostadmin delete <名称>` - 删除区域
- `/gostadmin info <名称>` - 查看区域信息
- `/gostadmin clear` - 清除当前选区

## 游戏流程

1. **设置游戏区域**
   ```
   /gostadmin setarea 领地名称
   ```

2. **玩家加入队列**
   ```
   /gost join
   ```

3. **开始游戏**（当有足够玩家时）
   ```
   /gostadmin start
   ```

4. **游戏进行**
   - 准备阶段：20秒
   - 游戏阶段：7分钟
   - 每分钟自动发放道具
   - 感染机制：鬼攻击人类

5. **游戏结束**
   - 人类全部被感染 → 鬼胜利
   - 时间结束仍有幸存人类 → 人类胜利
   - 自动分发奖金

## 配置文件

配置文件位于 `plugins/Gost/config.yml`

### 主要配置项
```yaml
# 游戏设置
game:
  duration: 420          # 游戏时长（秒）
  preparation-time: 20   # 准备时间（秒）
  queue-time: 60         # 队列等待时间（秒）
  min-players: 2         # 最小玩家数
  max-players: 16        # 最大玩家数
  max-games: 1           # 最大同时游戏数

# 经济设置
economy:
  entry-fee: 100.0       # 入场费
  server-bonus: 5000.0   # 服务器奖金

# 道具设置
items:
  adrenaline:
    duration: 10         # 肾上腺素持续时间
    speed-level: 2       # 速度等级
```

## 权限节点

- `gost.join` - 加入游戏（默认true）
- `gost.leave` - 离开游戏（默认true）
- `gost.admin` - 管理员权限（默认op）

## 注意事项

1. **领地设置**：必须先设置Residence领地才能开始游戏
2. **经济系统**：需要Vault和经济插件支持
3. **生命值**：需要Injured Effects模组来实现2点生命值效果
4. **PVP**：服务器PVP需要开启（用于感染判定）
5. **背包锁定**：游戏期间禁止丢弃和移动物品

## 故障排除

### 常见问题
1. **无法开始游戏**
   - 检查Residence领地是否设置
   - 检查等待玩家数量是否足够

2. **经济系统不可用**
   - 检查Vault和经济插件是否安装
   - 检查玩家是否有足够金币

3. **感染无效**
   - 检查服务器PVP是否开启
   - 检查玩家是否在游戏区域内
