插件概述

Gost 是一款融合了多种游戏模式的 Minecraft 服务器插件，结合了 猫抓老鼠、鬼抓人、躲猫猫、CF生化模式 的核心玩法。玩家在紧张刺激的对抗中体验人类与鬼之间的生死追逐，运用策略与道具成为最后的幸存者。

核心特色

紧张刺激的追逐体验

人类 vs 鬼，生死追逐，极限逃生。时间压力下的紧张对抗，感染转化机制带来动态阵营变化。

经济与奖励系统

入场费 + 奖池奖金，智能分配算法。高风险高回报，鼓励积极参与和策略性游戏。

智能匹配与队列系统

自动匹配玩家，满员加速倒计时。v2.2.0 新增队列人数不足自动清空功能。

多层次道具系统

初始道具 + 随机刷新道具，阵营限制 + 冷却机制。智能时间管理，关键时刻提示。

沉浸式视觉系统

Boss栏显示，ActionBar提示，居中大文本。鬼玩家粒子效果，黑暗效果，心跳声系统。

灵活区域管理系统

简易选区工具，多区域存档，一键管理。自动传送，边界保护，智能区域选择。

安装与配置

系统要求

  • Minecraft 版本：1.20.x（最佳兼容性）
  • 服务器核心：Paper/Spigot（推荐 Paper）
  • 必需插件：Vault + 经济插件（如 EssentialsX）
  • 可选模组：Injured Effects（增强恐怖氛围）

安装步骤

  1. 确保服务器已安装 Vault 和经济插件
  2. 将 Gost 插件放入 plugins/ 目录
  3. 重启服务器
  4. 使用 /gostadmin tool 获取选区工具
  5. 创建游戏区域（详见区域管理部分）

快速配置

# 基础配置示例
game:
  min-players: 2 # 最小玩家数
  max-players: 16 # 最大玩家数
  duration: 420 # 游戏时长（秒）
  queue-time: 60 # 队列等待时间

economy:
  entry-fee: 100.0 # 入场费
  server-bonus: 5000.0 # 服务器奖金

游戏玩法

游戏流程

  1. 区域设置：管理员使用岩浆膏选择游戏区域
  2. 加入队列：玩家使用 /gost join 加入游戏队列
  3. 自动匹配：达到最小玩家数后自动开始游戏
  4. 角色分配：随机分配人类/鬼角色
  5. 游戏进行：人类躲避感染，鬼追击感染
  6. 游戏结束：时间结束或所有人类被感染

角色系统

角色 描述 特殊能力
人类 幸存者阵营 使用道具，躲避感染
普通鬼 感染者阵营 感染人类，速度提升
母体鬼 初始感染者 前20秒禁足，高亮显示

感染机制

  • 感染方式：鬼玩家右键点击人类玩家
  • 感染条件：双方必须在游戏区域内
  • 感染效果：人类立即转变为鬼阵营
  • 感染提示：屏幕字幕 + 音效 + 闪电效果

命令系统

玩家命令

管理员命令

区域管理

命令 描述
/gostadmin tool 获取区域选择工具（岩浆膏）
/gostadmin pos1 设置第一个点（工具左键）
/gostadmin pos2 设置第二个点（工具右键）
/gostadmin save <名称> 保存当前选区
/gostadmin list 列出所有区域
/gostadmin load <名称> 加载指定区域
/gostadmin delete <名称> 删除指定区域
/gostadmin info <名称> 查看区域信息
/gostadmin clear 清除当前选区

游戏管理

命令 描述

/gostadmin stop 强制停止游戏
/gostadmin reload 重载配置文件
/gostadmin status 查看游戏状态
/gostadmin dark <on|off|status> 黑暗效果管理
/gostadmin heartbeat <on|off|status> 心跳声管理

权限节点

玩家权限（默认所有玩家拥有）

  • gost.player - 基础玩家权限（使用所有玩家命令）

管理员权限

  • gost.admin - 管理员基础权限
  • gost.admin.create - 创建区域权限
  • gost.admin.delete - 删除区域权限
  • gost.admin.reload - 重载配置权限
  • gost.admin.manage - 管理游戏权限
  • gost.admin.divineguardian - 管理神圣守护权限 v2.1.1
  • gost.admin.ghostparticle - 管理鬼玩家粒子效果权限 v2.1.2

配置详解

主要配置项

  duration: 420 # 游戏总时长（秒），默认7分钟
