🌑 黑暗效果与疾跑并存系统（v2.2.1 引入）

原生DARKNESS效果：使用Minecraft原生的DARKNESS效果（1.19+）或BLINDNESS效果（旧版本）。
疾跑功能修复：通过属性修改器为每个受黑暗效果的玩家增加30%移动速度加成。
智能检查机制：每10 ticks检查玩家疾跑状态，自动恢复被阻止的疾跑。
版本兼容性：自动检测并适配不同Minecraft版本的效果类型。

配置项：dark-effect.enabled: true（默认启用）。

🛡️ 神圣守护系统 & 猎魔人阶段增强功能（v2.2.2 新增）

1. 鬼玩家复活机制

猎魔人阶段鬼玩家被击杀后进入旁观模式，可配置的复活倒计时（默认15秒），倒计时显示在ActionBar上，复活时播放视觉效果和音效。

2. 智能血量调整系统

普通鬼血量降至2点，母体鬼血量降至3点，猎魔人血量固定为2点（均可配置）。自动适应游戏阶段变化。

3. 禁止回血机制

猎魔人阶段禁止所有玩家回血（包括自然恢复、饱食度、药水效果），并提供明确的游戏提示。

4. 收割者伤害系统

基于伤害值而非攻击次数的击杀逻辑，可配置每次攻击伤害值，显示伤害值和剩余血量信息。

配置项：divine-guardian.demon-hunter 下可调整复活时间、血量、伤害等参数。

🧰 完整道具图鉴
⚡ 感染机制 & 全局高亮效果

感染方式：鬼玩家 左键/右键点击 人类玩家进行感染。被感染人类立即转为鬼阵营，播放闪电/音效。
✨ 一次机会道具可被动抵挡一次感染。

🌟 道具发放全局高亮：每次发放道具时，所有存活玩家获得持续 5秒 的高亮效果。

🏆 经济与奖励系统
💰 奖池分配规则

入场费100金币 + 服务器奖金5000金币。人类胜利时人类70%奖池，鬼30%；鬼胜利时鬼100%奖池。个人按存活时间/感染人数分配。

🛡️ 神圣守护模式（传统 & 救赎者）
⚔️ 模式1：传统神圣守护

最后人类获得3次感染免疫，被攻击时随机传送，速度I+发光，次数耗尽后隐身10秒。

🔥 模式2：救赎者系统

最后人类成为救赎者，获得「神之救赎」道具，可转化鬼玩家回人类（默认2次），使用后随机传送，高亮+速度效果。命令 /divineguardian setmode 2。

👻 鬼转人类功能

游戏最后阶段（默认剩余3分钟），随机将非母体的鬼玩家转换回人类阵容。配置项：ghost-to-human.enabled: false（默认关闭）。

🎯 完整游戏流程
1. 区域设置
/gostadmin tool 选区 → save → load enable
2. 玩家准备
/gost join，队列满员10秒倒计时，人数不足自动清空并退还金币。
3. 游戏进行阶段
准备20秒，游戏7分钟，每分钟道具刷新（全体高亮5秒），母体鬼禁足20秒。
4. 猎魔人阶段（v2.2.2）
游戏剩余90秒时自动触发，最后2名人类变为猎魔人，鬼玩家血量调整，禁止回血，鬼被击杀后15秒复活。
5. 游戏结束
人类胜利：时间结束有人类存活；鬼胜利：所有人类被感染；猎魔人胜利：消灭所有鬼玩家。
⚙️ 配置文件 (config.yml)
📋 主要配置项示例 
# Gost 插件配置文件 v2.2.2
game:
  duration: 420
  min-players: 2
  max-players: 16
divine-guardian:
  enabled: true
  trigger-human-count: 2
  demon-hunter:
    phase-start-time: 90
    respawn:
      enabled: true
      respawn-time: 15
    health:
      ghost-normal: 2.0
      ghost-mother: 3.0
      demon-hunter: 2.0
      no-healing: true
dark-effect:
  enabled: true
