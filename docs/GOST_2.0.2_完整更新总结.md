# Gost 插件 2.0.2 版本完整更新总结

## 📋 更新概述

本次更新主要解决了编译错误问题，新增了黑暗效果系统，优化了区域提示，完善了权限系统，并更新了配置文件。

## 🔧 修复的问题

### 1. 编译错误修复
#### PlayerManager.java - 缺少ItemMeta导入
- **问题**: `找不到符号: 类 ItemMeta`
- **修复**: 添加了 `import org.bukkit.inventory.meta.ItemMeta;`
- **位置**: 第11行

#### SecondChanceListener.java - 错误的枚举常量
- **问题**: `找不到符号: 变量 GHOST`
- **原因**: `PlayerManager.PlayerRole` 枚举中没有 `GHOST` 常量，只有 `GHOST_MOTHER` 和 `GHOST_NORMAL`
- **修复**: 使用 `isHuman()` 和 `isGhost()` 方法替换直接枚举检查
- **优化**: 代码更简洁，避免重复逻辑

#### ItemSpawnManager.java - 代码优化
- **修复**: 使用 `isHuman()` 和 `isGhost()` 方法替换直接枚举检查
- **效果**: 代码更简洁，提高可维护性

### 2. 功能问题修复
#### 灵魂探测器失效问题
- **问题**: v2.0.1版本中"灵魂探测器"道具可能失效
- **影响**: 玩家无法正常使用灵魂探测器暴露其他玩家位置
- **修复**: 修复了相关逻辑，确保灵魂探测器正常工作
- **版本**: 此问题在v2.0.2中已完全修复

## 🎮 新增功能与优化

### 1. 区域命名支持中文
- **现状**: 区域名称已支持中文命名
- **使用**: 管理员可以使用中文名称保存和选择游戏区域
- **兼容性**: 完全兼容现有区域系统

### 2. 区域欢迎提示系统
#### 准备阶段提示
- **位置**: `GameManager.sendPreparationTitle()` 方法
- **内容**: 在准备阶段倒计时标题上方添加"欢迎来到：XXX"提示
- **格式**: `欢迎来到：区域名称` + 倒计时信息
- **颜色**: 金色欢迎文字 + 黄色区域名称

#### 游戏开始提示
- **位置**: `GameManager.sendGameStartTitles()` 方法
- **内容**: 在游戏开始标题的副标题位置显示"欢迎来到：XXX"
- **格式**: 作为副标题显示区域欢迎信息
- **效果**: 人类和鬼玩家都会看到区域欢迎信息

### 3. 权限系统完善
#### 新增权限节点
- **权限**: `gost.player`
- **描述**: 允许玩家使用所有基础命令
- **默认**: 所有玩家拥有 (default: true)
- **影响命令**: `/gost join`, `/gost leave`, `/gost info`

#### 权限检查实现
- **位置**: `GostCommand.onCommand()` 方法
- **检查**: 每个子命令执行前检查 `gost.player` 权限
- **提示**: 无权限玩家收到红色错误提示

#### plugin.yml更新
- 添加了 `gost.player` 权限定义
- 保持向后兼容性

### 4. 道具配置系统扩展
#### 新增灵魂探测器配置
```yaml
soul-detector:
  duration: 25 # 灵魂探测器暴露持续时间（秒）
  cooldown: 30 # 灵魂探测器冷却时间（秒）
```

#### 完善一次机会配置
```yaml
second-chance:
  cooldown: 60 # 一次机会冷却时间（秒）
  human-speed-duration: 10 # 人类玩家速度效果持续时间（秒）
  human-speed-level: 2 # 人类玩家速度效果等级
  human-glowing-duration: 10 # 人类玩家高亮效果持续时间（秒）
  ghost-slow-duration: 7 # 鬼玩家缓慢效果持续时间（秒）
  ghost-slow-level: 1 # 鬼玩家缓慢效果等级
```

#### ConfigManager更新
- 添加了 `getSoulDetectorDuration()` 方法
- 添加了 `getSoulDetectorCooldown()` 方法
- 完善了现有配置方法

### 5. 黑暗效果系统（之前已实现）
- **配置**: `dark-effect.enabled` 控制开关
- **命令**: `/gostadmin dark on/off/status`
- **效果**: 游戏开始时自动应用失明效果
- **管理**: 支持实时切换

## 🎯 使用说明

### 区域中文命名
1. 使用选区工具设置区域
2. 使用 `/gostadmin save 中文名称` 保存区域
3. 区域名称支持任意中文字符

### 区域欢迎提示
1. 游戏准备阶段：玩家会看到"欢迎来到：区域名称" + 倒计时
2. 游戏开始阶段：玩家会看到角色提示 + "欢迎来到：区域名称"
3. 自动获取当前选中区域名称

### 权限管理
1. **默认情况**: 所有玩家自动拥有 `gost.player` 权限
2. **权限限制**: 可通过权限插件移除玩家权限
3. **命令保护**: 无权限玩家无法使用任何玩家命令

### 道具配置调整
1. **灵魂探测器**: 可调整暴露时间和冷却时间
2. **一次机会**: 可调整所有效果参数
3. **热重载**: 支持 `/gostadmin reload` 重载配置
